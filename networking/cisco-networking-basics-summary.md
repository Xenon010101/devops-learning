# Cisco Networking Basics – Summary (My Learning)

I completed the Networking Basics course from Cisco Networking Academy.

This course helped me understand how communication actually happens between systems, which is important for cloud and DevOps.

## What I Learned

- How devices communicate over a network
- Difference between IP address and MAC address
- Basic idea of DNS and how domain names work
- Common protocols like HTTP, HTTPS, and SSH
- Roles of networking devices (router, switch, server)

## What Improved for Me

Before this, networking felt confusing. Now I can:
- Understand how a request goes from client to server
- Identify basic network components
- Visualize how data flows

Moving to Linux to understand how systems actually run in real environments.
